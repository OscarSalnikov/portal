Рекомендации по использованию

Правила для консистентного UI
Цвет: #6C5CE7 для основных CTA, #A29BFE для вторичных действий, #E6B0AA для теплых подсветок, #0F62FE для A11y/высокого контраста.
Кнопки: Основная — главное действие экрана; Вторичная — поддерживающее; Ghost — нейтральные ссылки в карточках.
Типографика: База 16px (18px в A11y), H1 32px для заголовков страниц, Caption 12px для статусов.
Отступы: 8/12/16/24/32px внутри карточек, 40–56px между секциями; держите вертикальный ритм.
Сетка: 12 колонок с gap 16px на desktop, auto-fit minmax(220–280px) для карточек, padding контейнера 24–56px.
Радиусы и тени: 14px для кнопок/полей, 16px для карточек; мягкая тень var(--shadow-soft) по умолчанию, отключайте тени в high-contrast.
Состояния: Hover — подъем и усиленная тень; Active — легкое нажатие; Disabled — сниженная прозрачность без анимации.
