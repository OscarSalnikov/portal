Usage Guidelines

Practical rules for consistent UI
Color: Use #6C5CE7 for primary CTAs, #A29BFE for secondary actions, #E6B0AA for warm highlights, #0F62FE for A11y/high-contrast accents.
Buttons: Primary for the main action on a screen; Secondary for supportive tasks; Ghost for neutral links inside cards.
Type: Base 16px (18px in A11y), H1 32px for page titles, Caption 12px for meta and statuses.
Spacing: Use 8/12/16/24/32px inside cards, 40-56px between sections; keep vertical rhythm consistent.
Grid: 12-column with 16px gap on desktop, auto-fit minmax(220-280px) for cards, container padding 24-56px.
Radius & shadows: 14px for buttons/fields, 16px for cards; soft shadow var(--shadow-soft) by default; remove shadows in high contrast.
States: Hover lifts with stronger shadow; Active adds light press; Disabled lowers opacity without animation.
